## Installation
### require
* pipenv

### install
`$ pipenv install`

### model file.
* download and extract-bz2 and place $ROOT/
    * http://dlib.net/files/shape_predictor_68_face_landmarks.dat.bz2


### how to execute
```
$ pipenv run python ./eye_blink_detector_dlib4.py
```
